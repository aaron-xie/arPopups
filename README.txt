弹窗插件jquery.arPopups.js
v1.0.1[2012.6.23]

1.可自定义触发事件（支持所有jq事件，如click、hover、keyup等等）；
2.在不触发click等事件的前提下显示弹窗方法：$.fn.arPopups({show:true})
3.支持设置弹窗定时关闭：如timeout:8000表示8s后自动关闭弹窗
4.支持ESC键关闭，或者在背景上双击即可关闭弹窗
5.元素class设为arp-close即可点击关闭弹窗，无论是ajax的html，还是隐藏的html，或者是自定义的html，甚至于iframe的内容中的.arp-close都能实现关闭功能
6.支持外部弹窗关闭方法：$.arPopups.close()
7.支持关闭后回调函数
8.自定义标题内容与窗口样式（外部CSS样式定制）；
9.可自定义窗口尺寸；
10.自定义遮罩层透明度；
11.可以根据需要添加弹窗标题
12.支持html内容自定义
13.支持在弹窗显示本页#id或.class里的内容
14.支持ajax页面内容，可自定义ajax路径（可设为某一属性值，如@href）
15.可对ajax目标内容进行筛选（id或class）
16.支持iframe页面调用
17.支持callback回调函数；
18.支持弹窗拖拽功能，drag为false关闭拖拽功能
19.可在外部设定插件默认参数
